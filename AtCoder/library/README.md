# rounded up

The quotient depends on whether there is a surplus.
`if(a % b == 0) return a / b`  
`if(a % b == 1) return (a / b) + 1`
If two expressions combine into one then
`(a - 1 / b) + 1`  
`(a + b - 1) / b`
